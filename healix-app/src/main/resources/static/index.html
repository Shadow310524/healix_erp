Welcome to Healix ERP Developer Console

This interface allows you to interactively test the multi-tenant aggregates, validation business rules, and PostgreSQL Row-Level Security (RLS) policies implemented in Sprint 1 and Sprint 2.

  • Tenant Isolation: Authenticating as different tenants restricts data display to that tenant only.
  • Role-Based Access: Only users with ROLE_TENANT_ADMIN can write/create resources. Standard users have read-only access.
  • Product Catalog Validations: Enforces pricing constraints: MRP >= PTR >= Purchase Rate and GST checks.
  • Supplier Validations: Enforces unique supplier names, PAN formats, and GSTIN lengths (exactly 15 chars).
